SentinelOne Endpoint Security

This integration requires a UTMStack agent to work properly. Please, make sure you have installed it before you continue.

  1. 1 Click on the "Settings" menu in the SentinelOne sidebar.

  2. 2 Go to "Integrations" -> "Syslog", and configure SentinelOne Endpoint Security to send logs to a UTMStack agent to ports:

    • 7012 TCP
    • 7012 UDP
    • 7062 TCP (TLS)

  3. 3 Enable log collector and this integration in the configuration file which you can find where your UTMStack Agent is located, in the path:

    {{paths.os}}

  4. 4 Click on the "Test" button to check connection with UTMStack, and save changes

  5. 5 Click on the button shown below, to activate the UTMStack features related to this integration
